# Product Spec

## Mission

Build a local-first assessment authoring tool that is easier than Brightspace for creating quizzes and tests from outside resources.

## V1 In Scope

- local-first project workflow
- plain text ingestion
- canonical assessment schema
- manual editor and review workflow
- Brightspace CSV exporter
- validation rules
- saved project files
- deterministic export behavior

## Later Phases

- DOCX ingestion
- PDF ingestion
- AI-assisted question generation
- answer-key inference with explicit warning states
- alternative exporters

## Out of Scope for V1

- Brightspace browser automation
- Brightspace API integration
- QTI export
- multi-user collaboration
