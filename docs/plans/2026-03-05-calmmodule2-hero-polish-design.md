# CALM Module 2 Hero Polish Design

**Date:** 2026-03-05

## Goal
Refine the interactive module's visual polish while preserving its playful violet clay identity.

## Scope
This pass focuses on:
- surface consistency
- section heading hierarchy
- support text readability
- spacing rhythm
- shared component polish

It does not change module structure, learning logic, or content flow.

## Direction
Keep the current violet clay visual language, but make it feel more intentional and premium by reducing inconsistency between cards, headers, and form surfaces.

## Design Moves
- Introduce a reusable section-header pattern with a small eyebrow, stronger title, and clearer supporting copy.
- Unify major activity cards around a more consistent clay treatment.
- Make support surfaces lighter and quieter than primary activity cards.
- Improve textarea/input feel through better typography, placeholder color, and focus depth.
- Refine `KnowledgeDrop` and `HintToggle` so they feel like part of the same system rather than isolated utility blocks.
- Tighten vertical rhythm so sections breathe more evenly.

## Expected Result
The module should still feel playful and approachable, but more composed and professional. Headings should guide the user better, surfaces should feel related, and the page should read as one designed experience instead of a stack of separate components.
